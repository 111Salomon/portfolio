À propos

Passionné par le développement web depuis mes débuts en informatique, je suis constamment à la recherche de nouvelles technologies et méthodes pour créer des solutions innovantes.

Mon approche allie esthétique moderne et fonctionnalité optimale, avec une attention particulière portée à l'expérience utilisateur et aux standards de qualité les plus élevés.

Toujours à l'affût des dernières tendances technologiques, j'accompagne mes clients dans la concrétisation de leurs visions créatives en solutions digitales durables et évolutives.

Expérience professionnelle

Système de pointage par code QR

Projet académique

Développement d’une solution de gestion des présences utilisant la technologie QR code : les employés scannent un code dynamique pour enregistrer leur arrivée et leur départ. Le système assure un suivi fiable, en temps réel, des horaires de travail, avec une interface simple et intuitif réalisé en JAVA.

Application web d’archivage foncier

Projet académique

Développement d’une application web permettant la gestion numérique et centralisée des documents fonciers (titres de propriété, actes, plans, etc.). L’outil facilite l’archivage, la recherche et la consultation sécurisée des dossiers grâce à une interface intuitive, développée avec PHP pour la logique métier et Bootstrap pour une expérience utilisateur responsive et moderne.

Application web de gestion des congés

Projet académique

Conception et développement d’une application web permettant la saisie, la validation et le suivi des demandes de congés du personnel. L’outil centralise les demandes, automatise les flux d’approbation et offre une vue claire de la disponibilité des équipes. Réalisé avec Laravel pour la robustesse du backend et MySQL pour la gestion fiable des données.

Application web de gestion de bibliothèque

Projet académique

Développement d’une application web pour la gestion des ouvrages, des emprunts et des adhérents, sans framework. Le backend a été entièrement codé en Python pur (avec un serveur HTTP personnalisé), tandis que le frontend repose sur JavaScript pour une interface interactive et dynamique. Ce projet met en œuvre les fondamentaux du développement web : routage manuel, gestion des requêtes HTTP, et manipulation du DOM.

Application mobile événementielle (prototype)

Orange digitale Center Madagascar

Projet de fin de formation (UI/UX Design)

Conception d’une application mobile centrée utilisateur pour la découverte, la gestion et l’interaction autour d’événements locaux (conférences, concerts, ateliers, etc.). À travers une démarche UX structurée — recherche utilisateur, personas, parcours utilisateurs, wireframes — j’ai défini une expérience fluide et intuitive, puis réalisé des maquettes haute fidélité et un prototype interactif sous Figma. L’interface met l’accent sur la clarté, l’accessibilité et l’engagement, en respectant les bonnes pratiques de design mobile (iOS/Android).
